# ESP32 30-Pin RC522 RFID Reader Development Board

This PCB is a simple development and testing board designed for building USB-type RFID reader products using an ESP32 30-pin development module and an RC522 RFID/NFC reader module.

The board is mainly created for quick prototyping, testing, and product development. It allows easy connection between the ESP32 and RC522 module, making it useful for RFID card reading, access control systems, attendance systems, vending machines, smart lockers, and other IoT-based RFID applications.

## Features

* Supports ESP32 30-pin development board
* Supports RC522 RFID/NFC reader module
* USB-powered through ESP32 board
* Easy to use for RFID reader product testing
* Compact design for prototype and development use
* Suitable for IoT and embedded product development
* Can be used as a USB-style RFID reader with custom firmware

## Applications

* RFID card reader
* Access control system
* Smart locker system
* Attendance system
* Vending machine user authentication
* IoT product testing
* ESP32 RFID development board

## Purpose

This board is designed as a development/test PCB to make RFID-based product development easier. It helps reduce wiring issues and allows faster testing compared to jumper wire connections.

## Notes

This is a prototype/development PCB. Firmware can be customized as per the final product requirement.

Built with ESP32 and RC522 technology, this project showcases the core functionality required for modern [personalized vending machines](https://digitalmonk.biz/custom-vending-machine-development/) and smart user authentication systems.
